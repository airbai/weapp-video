# 微信小程序－Alphabeille法语

### 说明：

实现alpha-beille.com网站的视频播放功能，特色：
- 使用wxParse解析库
- 实现按字幕播放视频
- 实现上拉加载，下拉刷新

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面文件
- utils — 存放格式化文件
- api — 存放数据文件
- wxParse — 存放html解析库文件

### 开发环境：

微信web开发者工具 v0.11.122100
